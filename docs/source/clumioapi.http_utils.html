clumioapi.http_utils.http_context module¶

class clumioapi.http_utils.http_context.HttpContext(request, response)[source]¶

Bases: object

A Context containing HTTP Request and Response.

An HTTP Context that contains both the original HttpRequest object that intitiated the call and the HttpResponse object that is the result of the call.

request¶

The original request object.

response¶

The returned response object after executing the request. Note that this may be None depending on if and when an error occurred.
